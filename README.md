Campus Event Management System
This project is a modern web application designed to help manage campus events easily.
 It is built using React, TypeScript, and Tailwind CSS and focuses on a clean user interface, smooth animations, and easy navigation.
The system allows students to view events, submit new events, and receive notifications, while administrators can manage and monitor all events from one dashboard.

Main Features
Core Features
Landing Page – Attractive first page with animations
Login System – Simple login using email, username, ID, and password
Event Dashboard – Displays all campus events with images
Event Submission – Users can submit events using a simple form
Admin Dashboard – Admins can view statistics, search events, and edit them
Notifications – Alerts for new, updated, or upcoming events
Contact Page – Allows users to contact the administration


 Event Categories
The system supports the following event types:
Technology
Conference
Cultural
Workshop
Training
Special Days


Technical Overview
React 18 with TypeScript
Tailwind CSS for styling
Motion (Framer Motion) for animations
Responsive design (works on mobile and desktop)
Real-time UI updates
Search and filter functionality
Inline editing in admin dashboard
