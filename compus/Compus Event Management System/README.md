
  # Compus Event Management System

  This is a code bundle for Compus Event Management System. The original project is available at https://www.figma.com/design/WZGENxcfyzDd4QdplsjKbX/Compus-Event-Management-System.

This project is a React + TypeScript web application developed for managing campus events.
It allows users to view events, submit events, receive notifications, and allows administrators to manage events.

The project is built using a component-based architecture and includes wireframes, reusable UI components, and functional pages.
